MicroBIN python library
=======================

MicroBIN is a compact binary serialization format.

`More info about MicroBIN
format <https://github.com/neko-neko-nyan/MicroBIN>`__


MicroBIN library provides json-like interface with ``load(s)`` and
``dump(s)`` functions.
